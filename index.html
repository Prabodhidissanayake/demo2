FizzBuzz Application

Game Rules:

  • Enter numbers between 1-100 separated by commas (Ex: 1,2,3,4,5,6)
  • Strings not accepted
  • Negative numbers are not accepted

You will see:

  • If the number is divide by 3 --> "Fizz"
  • If the number is divide by 5 --> "Buzz"
  • If the number is divide by both 3 & 5 --> "FizzBuzz"
